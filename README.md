# Dr PDF PRO

Browser-based PDF toolkit built with React + Vite + TypeScript.

## Requirements

- Node.js 20+
- npm 10+

## Run locally

```sh
npm install
npm run dev
```

## Scripts

- `npm run dev` — start development server
- `npm run build` — build production assets
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
- `npm run test` — run Vitest tests

## Tech stack

- React 18
- Vite 6
- TypeScript
- Tailwind CSS + shadcn/ui
- pdf-lib + pdfjs-dist
